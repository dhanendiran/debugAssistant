## ✨ Features

- 🔍 AI-powered error analysis  
- 🧾 Automatic keyword extraction from stack traces  
- 💡 Clear explanations of errors  
- 🛠 Suggested fixes and improvements  
- 📚 Debug history tracking (MongoDB)  
- ⚡ Fast REST API backend  
- 🌐 Modern React + Vite frontend

## 🛠 Tech Stack

| Layer      | Technology |
|-----------|------------|
| Frontend  | React (Vite), Axios |
| Backend   | Node.js, Express |
| Database  | MongoDB + Mongoose |
| AI Layer  | External AI API |

## ⚙️ How It Works

Error Input → Keyword Extraction → AI Analysis → Explanation + Fix → Store in DB → Show History
